# TsinghuaNews 清华动态新闻爬虫

一个用于学习与实践的新闻爬虫项目，目标站点为 **清华大学官网「最新动态」栏目**。  
项目涵盖了列表页抓取、新闻详情页解析、文本与图片保存、日志统一管理，以及后续的文本分词与词频统计分析。

> 本项目主要用于课程作业与个人学习实践。

---

## 功能简介

- 抓取新闻列表页，获取新闻标题、链接、日期
- 逐条抓取新闻详情页
- 每条新闻独立保存：
  - `content.txt`：新闻正文文本
  - `images/`：新闻内图片
- 统一日志系统（`log_manager`）：
  - 控制台输出
  - 文件日志输出（`logs/`）
- 支持 Debug 模式限制最大页数
- 对已抓取新闻文本进行 **中文分词与词频统计**
  - 输出出现次数最多的前 N 个词语

## 项目结构说明

```text
TsinghuaNews/
├── run.py                  # 项目入口

├── news_detail.py          # 新闻详情页解析与保存

│   └── settings.py         # 项目配置
├── output/
│   └── news/               # 新闻数据（每条新闻一个文件夹）
├── logs/                   # 日志输出目录
├── .env                    # 环境变量（不提交）
├── .gitignore
└── README.md


```
